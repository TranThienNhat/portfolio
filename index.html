Giới thiệu

Xin chào! Tôi là một Full-stack Developer mới tốt nghiệp, có đam mê với việc phát triển hệ thống backend hiệu quả và giao diện frontend hiện đại.

Tôi đã hoàn thành các khóa học CS50x và CS50W của Harvard, cung cấp cho tôi nền tảng vững chắc về khoa học máy tính và phát triển web. Với kinh nghiệm phát triển cả backend Java Spring Boot và frontend Angular, tôi có thể xây dựng các ứng dụng web đầy đủ từ database đến giao diện người dùng.

Tôi đang tìm kiếm cơ hội để áp dụng và mở rộng kỹ năng của mình trong môi trường phát triển phần mềm năng động, nơi tôi có thể đóng góp vào việc xây dựng các giải pháp công nghệ toàn diện và hiệu suất cao.

Dự án

E-Commerce Backend

Java 17 Spring Data JPA Spring Boot 3.2 Spring Security 6.1 MySQL 8 JUnit 5 Docker Swagger 3.0

Xây dựng hệ thống backend hiện đại cho nền tảng thương mại điện tử với kiến trúc RESTful API và các tính năng bảo mật cao cấp. Dự án này ứng dụng những tính năng mới nhất của Java 17 và Spring Boot 3.2 để cung cấp nền tảng vững chắc cho ứng dụng thương mại điện tử đa nền tảng.

Thành tựu chính:
  • Thiết kế và triển khai kiến trúc MVC sử dụng Spring Boot 3.2 với việc tách biệt các dịch vụ: xác thực, quản lý sản phẩm, giỏ hàng, đơn hàng và gửi email
  • Triển khai Spring Security 6.1 với xác thực JWT, hệ thống phân quyền chi tiết (RBAC) và xử lý refresh token an toàn
  • Phát triển API RESTful tuân thủ chuẩn HATEOAS với tài liệu OpenAPI 3.0 (Swagger) tích hợp
  • Đạt độ bao phủ unit test trên 85% với JUnit 5 và Mockito

Furniture Showroom Frontend Application

Angular 19 TypeScript Angular CLI Bootstrap RxJS ASP.NET HTML5 CSS3

Phát triển ứng dụng frontend hiện đại sử dụng Angular 19 với kiến trúc component-based và tích hợp với backend ASP.NET. Dự án này thể hiện khả năng xây dựng giao diện người dùng tương tác cao và responsive design với các công nghệ frontend tiên tiến nhất.

Đặc điểm nổi bật:
  • Xây dựng Single Page Application (SPA) với Angular 19 và Angular CLI phiên bản mới nhất, tối ưu hóa hiệu suất và trải nghiệm người dùng
  • Tích hợp Bootstrap framework để tạo giao diện responsive và mobile-friendly với thiết kế hiện đại
  • Sử dụng TypeScript và RxJS để quản lý state và xử lý bất đồng bộ một cách hiệu quả
  • Kết nối với backend ASP.NET thông qua RESTful API để trao đổi dữ liệu real-time
